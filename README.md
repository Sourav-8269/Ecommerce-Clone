
# Project Nykaa Clone

An e-commerce website for clothes and accessories.

Live:https://nykaaone.vercel.app/

## Features

 - Responsive Navbar
 - Some carousels
 - Login and SignUp
 - Products page which allows to buy Products
 - Add to cart page which shows products added in cart
 - Payment page
